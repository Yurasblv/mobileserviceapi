### All command for setting up and down project are in 
>`Makefile`
 
To run this project in development,
you will need to add the following 
environment variables to your .env file:
>`SECRET_KEY`
`POSTGRES_USER`
`POSTGRES_PASSWORD`
`POSTGRES_DB`
`POSTGRES_PORT`
`POSTGRES_HOST`
